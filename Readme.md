1️ Run Backend

Open PowerShell

Go to your backend folder:

cd D:\university-chatbot\Backend

Start backend with nodemon:

npm run dev

You should see something like:

Connected to MySQL database
Server running on port 3000
API Base URL: http://localhost:3000



Keep this terminal open.

2️ Run Frontend
Start the Frontend and run this: live-server

Note:

Install Live Server extension (if not installed)

Or in this way:
Right-click index.html → Open with Live Server or
Run it using the terminal: live-server

This will open a browser URL like:

http://127.0.0.1:5500
